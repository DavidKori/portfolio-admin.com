## Sections

1. **Profile** - Personal information and images
2. **About** - Bio and key highlights
3. **Skills** - Technical and professional skills
4. **Projects** - Portfolio projects with media
5. **Experience** - Work history
6. **Education** - Academic background
7. **Certifications** - Professional certifications
8. **Achievements** - Awards and recognitions
9. **Blogs** - Articles and blog posts
10. **Testimonials** - Client and colleague feedback
11. **Resume** - Downloadable resume/CV
12. **Contact** - Contact information
13. **Social Links** - Social media profiles
14. **Messages** - Visitor messages

## Tech Stack

- **Frontend**: React 18, React Router 6
- **State Management**: React Context API
- **HTTP Client**: Axios
- **Styling**: CSS Modules with CSS Variables
- **Icons**: Emoji icons (can be replaced with icon library)
